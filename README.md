# Eagle下のプロダクツ

## Board1
ESP-WROOM-02と安定化電源を搭載した標準基板です。「さくっと使えるESP-WROOM-02基板」がコンセプトです。

## Board1 DustCounter
Board1をベースにして、ダストカウンターPPD42とOLED（I2c, 0.96インチのあれ）を簡単に接続して使えるようにしました。
